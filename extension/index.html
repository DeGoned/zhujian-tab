      系统通知须知
      • Chrome 必须在运行才能弹通知（后台进程在即可，无窗口也行）
      • macOS 系统设置 → 通知 → Google Chrome 必须允许

      • 错过的提醒会在 Chrome 下次启动时聚合通知（最多回溯 7 天）
      • chrome.alarms 不保证准时，可能延后 1-2 分钟
